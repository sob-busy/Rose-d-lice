🚀 DÉMARRAGE IMMÉDIAT (5 MINUTES)
===================================

1️⃣ Configuration Firebase
   - Allez sur: https://console.firebase.google.com
   - Créez un projet "rose-delice"
   - Activez Firestore Database
   - Générez clé de service
   - Remplissez backend/.env avec les données

2️⃣ Backend
   - Ouvrez Terminal 1
   - cd backend
   - npm install
   - npm run dev
   - Attendez: "Serveur Rose Délice en cours..."

3️⃣ Frontend
   - Ouvrez Terminal 2
   - cd frontend
   - npm install
   - npm run dev
   - Attendez: "VITE v5.0.8 ready"

4️⃣ Ouvrir le site
   - http://localhost:3000 🌐
   
🎉 VOILÀ! Votre site est vivant!
