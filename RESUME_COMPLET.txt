🌹 ROSE DÉLICE - SITE WEB PÂTISSERIE
=====================================

Projet complet de site web pour pâtisserie artisanale à Lomé, Togo

📊 RÉSUMÉ RAPIDE
================

✅ 30+ fichiers créés
✅ Frontend React + Vite (8 composants)
✅ Backend Node.js + Express (6 routes API)
✅ Database Firebase Firestore (2 collections)
✅ ~3000 lignes de code commenté en FRANÇAIS
✅ Animations fluides Framer Motion
✅ Responsive Mobile/Tablette/Desktop
✅ Palette Rose poudré + Crème
✅ Intégration WhatsApp directe
✅ Upload photos + Système d'avis

STRUCTURE PRINCIPALE
====================

frontend/
  ├── React 18 + Vite (ultra-rapide)
  ├── Tailwind CSS (styles élégants)
  ├── Framer Motion (animations fluides)
  └── 8 composants:
      - Navbar (navigation fixe)
      - Hero (accueil)
      - Gallery (grille photos)
      - Reviews (avis clients)
      - About (à propos)
      - Contact (coordonnées)
      - FloatingWhatsApp (bouton flottant)
      - Footer (pied de page)

backend/
  ├── Express.js (serveur web)
  ├── Firebase Firestore (base de données)
  ├── Multer (upload fichiers)
  └── 3 routes API:
      - GET/POST /api/gallery
      - GET/POST /api/reviews
      - POST /api/upload

DÉMARRAGE RAPIDE (5 MINUTES)
===========================

1. Installer Node.js: https://nodejs.org

2. Configuration Firebase:
   ✓ Créer projet: https://console.firebase.google.com
   ✓ Activer Firestore Database
   ✓ Générer clé de service
   ✓ Remplir backend/.env

3. Terminal 1 - Backend:
   $ cd backend
   $ npm install
   $ npm run dev
   → http://localhost:5000 ✅

4. Terminal 2 - Frontend:
   $ cd frontend
   $ npm install
   $ npm run dev
   → http://localhost:3000 ✅

API ENDPOINTS
=============

Galerie:
  GET  /api/gallery              → Photos approuvées
  POST /api/gallery              → Ajouter photo (en attente)
  DEL  /api/gallery/:id          → Supprimer photo (admin)

Avis:
  GET  /api/reviews              → Avis approuvés
  POST /api/reviews              → Ajouter avis (en attente)
  GET  /api/reviews/rating/average → Note moyenne

Upload:
  POST /api/upload               → Upload image → retourne URL

DATABASE (FIREBASE FIRESTORE)
==============================

Collections:
  
  gallery:
    - titre (string)
    - description (string)
    - image_url (string)
    - categorie (string)
    - note (number)
    - date_ajout (timestamp)
    - approuve (boolean) ← importante!

  reviews:
    - nom (string)
    - email (string)
    - produit (string)
    - note (number 1-5)
    - commentaire (string)
    - date (timestamp)
    - approuve (boolean) ← importante!

⚠️ NOTE: Photos et avis sont approuvés avant affichage
(approuve: false par défaut)
